Blogger Cheat Sheet Indonesia

Cheat Sheet ini bersumber dari btemplates.com

  • #Comment
  • #Header

Layout

  • #navbar-iframe - area NavBar Blogger
  • #outer-wrapper - Bungkus luar untuk membungkus keseluruhan halaman
  • #header-wrapper - Membungkus bagian header, title, dan description
  • #content-wrapper - Membungkus bagian main dan sidebar
  • #main-wrapper - Membungkus bagian blog post
  • #sidebar-wrapper - Membungkus bagian sidebar
  • #footer-wrapper - Membungkus bagian footer
Ini adalah pembungkus yang lebih besar, di dalamnya kita bisa menemukan satu atau lebih area gadget

Posts (Loop)

  • .post - Pembungkus area Post
  • .post-title - Judul Post
  • .post-header-line-1 - informasi di bawah judul post seperti tanggal, jam, nama penulis, dan lain-lain
  • .date-header - tanggal dimana Post dipublish
  • .post-author - menampilkan post penulis
  • .post-timestamp - waktu dimana Post dipublish
  • .post-label - daftar label Post
  • .comment-link - link ke Post Komentar
  • .post-backlinks - Trackbascks Post
  • .star-ratings - System rangking Star-ratins
  • .reaction-buttons - Evaluasi oleh reaction system
  • .post-location - Post Geolocation
  • .post-body - Post Body, Artikel
  • .post-footer - Informasi footer seperti tags, author, dan lain-lain
  • #blog-pager - mengandung link navigasi (post lama / older entries dan post terbaru / newer entries)
  • .blog-page-older-link - link navigasi older entries
  • .blog-page-newer-link - link navigasi newer entries
  • .home-link - Link navigasi Home
  • .feed-links - link subscription ke "Post's (Atom)
Gaya utama yang harus dipertimbangkan di setiap posting, sebagian besar tanggal tidak ditampilkan sampai pengguna memasukkannya dari panel

Comment

  • #comments - Pembungkus Comment dan comment-form
  • #comments h4 - judul komentar
  • dl.comments-block - pembungkus Comment block
  • .comment-author - link Comment's author dan icon
  • .comment-body - Comment's body
  • .comment-footer - Comment's footer
  • .comment-timestamps - Comment's timestamp (waktu publish komentar)
  • .comment-form - pembungkus Comment-form
  • h4.comment-post-message - judul Comment-form
Gaya untuk daftar dan bentuk komentar

Status Bar

  • .busc.status-msg-wrap - Status Bar
  • .status-msg-body - pesan status bar
  • .status-msg-border - border status bar
Status bar, ditunjukkan dengan seraching atau memasukkan tag

Gadget

  • .header - area block yang hanya memiliki satu widget: header (title dan description)
  • .main - area block yang hanya memiliki satu widget: the entries
  • .sidebar -
    • Zone edible pada sidebar untuk menambahkan / menghapus widget
  • .footer -
    • Zone edible di foot halaman untuk menambahkan/ menghapus widget
Area ini mungkin berisi gadget, "main" dan "header" hanya memiliki satu gadget dan terkunci secara default

Header

  • #header-inner - bungkus lapisan dalam untuk Header, title, dan description
  • h1.title - Judul Blog
  • p.description - deskripsi Blog
Gaya utama di dalam header

Gadget/Widgets

  • .widget - Mengandung beberapa widget (sidebar, footer, atau main)
  • .widget-content - content widget
  • .widget h2 - judul widget
  • .Label - daftar tag pada widget
  • .BlogArchive - widget mengandung arsip blog
  • .Poll - widget Polling
  • .VideoBar - widget Videobar
  • .SlideShow - widget Slideshow
  • .Image - widget Image
  • .AdSense - widget AdSense
  • .Text - widget Text
  • .HTML - widget HTML code
  • .CustomSearch -widget Custom search
  • .Followers - widget Follower (via Google Friend Connect)
  • .Profile - widget profile Author
  • .BloggerButton - tombol "Powered by Blogger"
  • .Subscribe - widget link Subscription-lis
  • .NewsBar - widget bar Google News
  • .Feed - daftar link RSS URL
  • .LinkList - widget LinkList
  • .BlogList - widget BlogRoll (daftar blog)
Semua widget memiliki skema serupa, tapi mereka memiliki satu kelas lagi bergantung pada jenisnya
