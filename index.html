Blogger Cheat Sheet Russia

Список id и классов, используемых в стандартных и кастомных шаблонах Blogger.
  • Общие блоки
  • Комментарии
  • Заголовок
  • Записи
  • Инфопанель
  • Гаджеты/виджеты

Общие блоки

  • #navbar-iframe - панель навигации Blogger
  • #outer-wrapper - блок для всего блога
  • #header-wrapper - блок для верхней части блога
  • #content-wrapper - блок для записей и боковой колонки
  • #main-wrapper - блок для записей
  • #sidebar-wrapper - блок для боковой колонки
  • #footer-wrapper - блок для нижней части блога
  • .header - область для гаджетов в верхней части блога
  • .main - область гаджета записей блога
  • .sidebar - область для гаджетов в боковой колонке
  • .footer - область для гаджетов в нижней части блога

Комментарии

  • #comments - блок для комментариев и формы для их добавления
  • #comments h4 - заголовок со количеством комментариев
  • dl.comments-block - блок для одного комментария
  • .comment-author - автор комментария
  • .comment-body - текст комментария
  • .comment-footer - нижняя часть комментария с дополнительной информацией
  • .comment-timestamps - дата и время публикации комментария
  • .comment-form - блок для формы добавления комментариев
  • h4.comment-post-message - заголовок перед формой добавления

Записи

  • .post - блок для записи
  • .post-title - заголовок записи
  • .post-header-line-1 - информация о записи под заголовком (дата и время публикации, автор, местоположение, прочее)
  • .date-header - дата публикации
  • .post-author - автор записи
  • .post-timestamp - время публикации
  • .post-label - ярлыки, указанные для записи
  • .comment-link - ссылка на форму для добавления комментариев
  • .post-backlinks - отслеживание записи
  • .star-ratings - рейтинг записи
  • .reaction-buttons - оценка зиписи
  • .post-location - местоположение записи
  • .post-body - текст записи
  • .post-footer - информация о записи под контентом (ярлыки, автор, местоположение, прочее)
  • #blog-pager - блок для ссылок навигации между записями
  • .blog-page-older-link - ссылка на старые записи
  • .blog-page-newer-link - ссылка на новые записи
  • .home-link - ссылка на домашнюю страницу
  • .feed-links - ссылки подписки на новые записи (Atom, RSS)

Инфопанель

  • .status-msg-wrap - блок для инфопанели
  • .status-msg-body - текст инфопанели
  • .status-msg-border - блок с границей инфопанели

Заголовок

  • #header-inner - блок для виджета заголовка блога
  • h1.title - заголовок блога
  • p.description - описание блога

Гаджеты/виджеты

  • .widget - блок для всего виджета
  • .widget-content - содержиоме виджета
  • .widget h2 - заголовок виджета
  • .Label - виджет списка ярлыков
  • .BlogArchive - виджет архива
  • .Poll - виджет опроса
  • .VideoBar - виджет видео
  • .SlideShow - виджет слайдшоу
  • .Image - виджет изображения
  • .AdSense - виджет AdSense
  • .Text - виджет текста
  • .HTML - виджет HTML
  • .CustomSearch - виджет поиска
  • .Followers - виджет подписчиков (vGoogle Friend Connect)
  • .Profile - виджет профиля автора
  • .BloggerButton - виджет кнопки "Powered by Blogger"
  • .Subscribe - виджет кнопок подписки на блог
  • .NewsBar - виджет для "Google Новости"
  • .Feed - виджет ссылки подписки на блог
  • .LinkList - виджет списка ссылок
  • .BlogList - виджет списка ссылок на блоги
